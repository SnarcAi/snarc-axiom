# Security Policy

SNARC AXIOM is a **research-only** repository (paper + protocol specification).  
It contains **no production code** and **no deployed network**.

## Reporting a security issue
If you believe you found a security issue in the **specification**, **threat model**, or **paper**:

1. Please open a GitHub Issue with a clear description, **or**
2. If you prefer private disclosure, contact the maintainers via the address listed in `COMMERCIAL_LICENSE.md`.

Include:
- a short summary
- affected file/section
- impact assessment (what could break)
- suggested mitigation (if any)

## Scope
In-scope:
- specification-level vulnerabilities
- cryptographic / protocol design issues
- safety/liveness/consistency issues

Out of scope:
- production deployment security (no production deployment exists here)
- exchange listings, tokens, fundraising
